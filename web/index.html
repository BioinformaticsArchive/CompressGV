CompressGV


...is browser-based and desktop software for the classification of novel non-synonymous single-nucleotide polymorphisms (nsSNPs).

This website will be extended to include more information after manuscript publication. Please use the example data by clicking the relevant links to copy into the input boxes.

Use CompressGV

  • Sequence Alignment
  • Deleterious nsSNPs
  • Neutral nsSNPs
  • Novel nsSNPs
  • Run CompressGV
  • Results

Alignments can be in FASTA format or with each species on a single line with no additional characters.
